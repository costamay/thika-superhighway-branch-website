# Thika Superhighway — Gor Mahia Green Army Branch

An **unofficial** fan branch website for the **Thika Superhighway** Green Army branch. It keeps members along the Thika Road corridor updated on Gor Mahia news, branch meetups, and match-day plans.

**Not** the official [Gor Mahia FC](https://www.gormahia.com) site.

## Sections

**Landing (`/`)**
- Hero — branch identity & corridor coverage
- About — branch mission, vision, and who we support
- Gallery — branch watch parties, travel, and fan moments
- Committee — branch volunteers (chair, secretary, transport, etc.)
- Footer with official club link

**Updates (`/blog`)**
- Gor Mahia fixtures with branch travel/watch-party notes
- Blog — branch announcements and curated K'Ogalo updates

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Stack

- React + Vite
- React Router

Edit copy and member data in `src/data/content.js`.
